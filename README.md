## Sobre Mim

Atuo estrategicamente no ciclo completo de desenvolvimento de software (SDLC), projetando e implementando soluções corporativas end-to-end. Minha especialidade é unir engenharia de software e infraestrutura para entregar aplicações escaláveis e de alto desempenho que atendem diretamente às necessidades do negócio.

- **Engenharia de Software:** Arquitetura de sistemas robustos e APIs utilizando Python (Django, FastAPI, Flask) no backend e ecossistema React/JavaScript no frontend.
- **DevOps & Infraestrutura:** Gestão autônoma de servidores Linux (on-premise e Cloud AWS/GCP), containerização com **Docker** e deploy de ambientes de produção.
- **Dados & Automação:** Desenvolvimento de robôs para automação de processos (RPA) e integração de dados corporativos utilizando PostgreSQL e Google BigQuery.
